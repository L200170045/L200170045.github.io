Untuk pengertian DDL dan DML sendiri yaitu, jika DDL (Data Defition Language) adalah kumpulan perintah SQL yang dapat anda gunakan untuk mengelola, mengubah struktur datatype dari objek pada database seperti index, table, trigger, view dan lain sebagainya.

Sedangkan DML (Data Manipulation Language) sendiri adalah sebuah kumpulan perintah yang berfungsi untuk pengelolaan database seperti membuat table, membuat database baru.
